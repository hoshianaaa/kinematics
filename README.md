# arc_angle.py

直径dの円の円周上をlだけ進んだとき始点と終点のなす角度

## input

```
d : diamitor (円の直径)
l : length (長さ)
```

## output

angle (角度) (/deg)


# cos 計算

## input

```
angle (角度) (/deg)
```

## output

output = cos(angle)

## examples

```

$ python3 cos.py 30
0.8660254037844387

$ python3 cos.py 90

$ python3 cos.py 180

$ python3 cos.py 270

$ python3 cos.py 360

```
